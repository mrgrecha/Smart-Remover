# Smart Remover. Console version. #
## How to install? ##
* $ python setup.py install
* $ source autocomplete.sh

## How to use? ##
### Removing ###
* $ srm 
### Working with trash bin ###
* $ trash
### Work with config ###
* $ config
### Cancel last operation ###
* $ undo
# For documentation add '-h' #

# Modes: #
* Dry Run
* Silent
* Force
